📚 Exam Pressure – Student Productivity Web App

A modern web application designed to help students manage exam stress, stay focused, and track their preparation effectively.

Exam Pressure is built to support students who feel overwhelmed during exams by providing tools that organize study routines, reduce pressure, and improve productivity.

🚀 Overview

Many students struggle with exam stress, poor planning, and last-minute preparation.
The Exam Pressure App aims to solve this by offering a structured platform where students can manage their tasks, stay organized, and maintain focus.

This project focuses on simplicity, performance, and user experience.

✨ Key Features

📋 Task Management – Organize study tasks and track progress

⏱ Focus & Productivity Tools – Helps students maintain study sessions

🔐 Secure Authentication – User login & registration system

🌐 Google Authentication – Quick login using Google account

📱 Responsive Design – Works smoothly on desktop and mobile

⚡ Fast Performance – Optimized frontend and backend
